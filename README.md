# web 基础模板

- [x] react
- [x] ant

## 运行和打包

```
$ yarn
$ yarn start
$ yarn sit // 切换sit环境启动
$ yarn build // 打包
$ yarn lint // eslint检查
```

## 命令操作

$ yarn new <component-name> // 新增组件
$ yarn delete <component-name> // 删除组件
